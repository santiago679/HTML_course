Headings tags

The Heading tags goes from H1 to H6, being the lower the most important and the last(6) the less important. Every Heading tag has a default size but we can change it with the style tag or using the css file linked to the HTML

Paragraph tag

The paragraph tag (p) is one of the most used tags when you write html. It allows you to write blocks of text. When we are writing text into an p tag, the browsers ignores additional spaces or empty lines, making a uniform and continue line of text
